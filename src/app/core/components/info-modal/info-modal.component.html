Про програму

Даний програмний продукт являє собою PWA (прогресивний веб-додаток), що дозволяє встановлювати його як нативний додаток на десктопні та мобільні платформи. Програму розроблено з використанням JavaScript фреймворку Angular v12.0.1. Шифрування реалізовано на основі алгоритму RC4.

Інструкція використання

В програми доволі простий та зручний інтерфейс. На сторінці присутні 3 поля:

  • (1) Поле для вводу посилання на рисурс для якого генерується пароль
  • (2) Поле для введення секретного слова (літери, цифри, спец сивмовли)
  • (3) Поле в яке буде внесено результуючий згенерований пароль

Для того щоб згенерувати пароль потрібно ввести дані та натиснути на кнопку "Генерувати" (4)

Також є можливість задати параметри генерації. Для цього потрібно натиснути на settings в правому верхньому куті сторінки, після чого з'явиться панель налаштувань:


  • Латиниця - літери латинського алфавіту (abc),
  • Кирилиця - літери кирилиці (абв),
  • Цифри - числа від 0 до 9,
  • Верхній регістр - використання Великих літер
  • Нижній регістр - використання маленьких літер
  • Спецсимволи - #/@!?$...
